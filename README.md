# Linux-snippets

## Description 

Application allows you to insert frequently used text by using simple snippet with parameters.

## Features

* Snippet parameters
* Snippet editor
* Snippet auto-completing

## Dependencies

* Python3.4
* PyGtk+ 3

* python-xlib
* xclip
* pyperclip
* libkeybinder-3.0.0
* gir1.2-keybinder-3.0
